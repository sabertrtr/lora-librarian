Set up your LoRA library

⚙ Categories 🗃 Collection 📚 Library

Point the librarian at the folder where your LoRA .safetensors files live. It scans locally in your browser and only ever sends a file's hash to the server — never the file itself.

Pass 1 — build the list (fast)

Pick your LoRA folder. Every .safetensors is listed by name and gets a card immediately — an uncategorized inventory of what you have.

Pass 2 — match to Civitai

Each file is read and SHA-256 hashed in your browser, then looked up on Civitai to fill in its image, name and tags. Cards populate one by one as it works. This can take a LONG time for large folders — it reads and hashes every file. You can leave it running.
